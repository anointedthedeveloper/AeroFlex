## Overview

AeroFlex is a sleek, performance-focused sneaker brand website built with modern web technologies. The site showcases a collection of premium athletic footwear with a cyberpunk-inspired design aesthetic featuring glassmorphism effects, gradient accents, and smooth animations.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Modern UI/UX**: Glassmorphism effects, gradient text, and smooth transitions
- **Product Showcase**: Interactive product cards with hover effects, quick view, and add-to-cart functionality
- **Mobile Navigation**: Hamburger menu with smooth slide-in animation for mobile devices
- **Newsletter Integration**: Email subscription forms for marketing campaigns
- **Social Media Links**: Connected social media presence across major platforms
- **Performance Optimized**: Fast loading times with CDN-hosted libraries

## Tech Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Custom properties, flexbox, grid, animations
- **JavaScript (ES6+)**: Interactive features and DOM manipulation
- **Bootstrap 5.3.2**: Responsive grid system and utilities
- **Tailwind CSS**: Utility-first CSS framework
- **Font Awesome 6.5.1**: Icon library
- **Google Fonts**: Poppins and Inter font families

## Installation

1. Clone the repository:
```bash
git clone https://github.com/anointedthedeveloper/AeroFlex-Performance-Sneakers.git
```

2. Navigate to the project directory:
```bash
cd AeroFlex-Performance-Sneakers
```

3. Open `index.html` in your browser or use a local development server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

4. Visit `http://localhost:8000` in your browser

## Usage

### Navigation
- Click the hamburger menu icon on mobile devices to access navigation
- Use the main navigation bar on desktop to switch between pages

### Product Interaction
- Hover over product cards to see the quick view button

- Click product cards to view detailed information
